# US Sanctions Zhejiang Monitoring v0.1.0

This bundle provides the same skill in several common agent-compatible formats.

## Skill Name

`us-sanctions-zhejiang-monitoring`

## Version

`0.1.0`

## Install Targets

### Codex

Copy:

```text
codex/us-sanctions-zhejiang-monitoring
```

to:

```text
~/.codex/skills/us-sanctions-zhejiang-monitoring
```

### Claude Code / Cloud Code-style SKILL.md Agents

Copy either:

```text
claude-code/us-sanctions-zhejiang-monitoring
```

or:

```text
cloud-code/us-sanctions-zhejiang-monitoring
```

into the agent's configured skills directory.

### Work Buddy

Import or paste:

```text
work-buddy/skill.md
```

If the tool supports JSON metadata, also import:

```text
work-buddy/skill.json
```

### Generic Agents

Use:

```text
universal/US_SANCTIONS_ZHEJIANG_SKILL.md
```

as a project instruction, custom skill, or system prompt addendum.
